# ChiChat

A modern chat application built with clean architecture and intuitive design.

## Features

- Real-time messaging
- User authentication
- Message history
- Responsive UI

## Installation

```bash
git clone https://github.com/yourusername/chichat.git
cd chichat
npm install
```

## Usage

```bash
npm start
```

Open your browser and navigate to `http://localhost:3000`

## Technologies

- Node.js
- Express
- React
- MongoDB
